Installera Chrome Tillägg Se flera avdelingar

När du klickar på "Lägg till ny avdelning" kommer du tilldelas ett unikt ID, som enbart är för dig och din avdelning. Detta ID sparas på servern för att kunna hänvisa till eran information. Skulle ni tappa bort ID kommer informationen inte att kunna återskapas. Vidare sparar servern i vart ID val av färg när registreringar genomförs, men inte vem som klickade eller när. Servern sparar slutligen specificerat ämne för var månad, för att ge mening till registreringarna.

Ny avdelning

Om du önskar få ID på din mail, skriv den nedanför, annars är det bara att lämna den tom. Göm bara inte att spara det ID du tilldelas.

Installera Google Chrome Tillägg

  1. Ladda ner ".crx" filen. Högerklicka och välj "Spara länk som...".
    extension.crx
  2. Öppna "chrome://extensions/" i ny flik.
  3. Aktivera Programmerarläge, uppe i högra hörnet.
  4. Dra in ".crx" filen till sidan.
  5. Klicka på "Lägg till tillägg".
