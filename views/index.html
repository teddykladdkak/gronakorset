Välkommen till Gröna korset!

När du klickar på "Lägg till ny avdelning" kommer du tilldelas ett unikt ID, som enbart är för dig och din avdelning. Detta ID sparas även på servern för att kunna hänvisa till er information, inget mer sparas för att kunna identifiera er. Det innebär att skulle ni tappa bort ID, kommer informationen inte att kunna återskapas. Vidare sparar servern i vart ID val av färg i Registrering, men inte vem som klickade eller när.

Om du önskar få ID på din mail, skriv den nedanför.
